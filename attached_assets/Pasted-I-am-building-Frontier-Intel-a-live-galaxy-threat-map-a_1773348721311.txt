## WHAT I AM BUILDING

Frontier Intel is an external web application (not an in-game mod) that 
connects to the EVE Frontier live universe through their blockchain 
gateway and displays real-time intelligence to players. Think of it as 
a war room dashboard — players open it before logging in to understand 
where danger is, where structures are, and what other players are doing.

---

## CORE FEATURES

1. LIVE THREAT MAP
   - A 2D interactive galaxy map showing all solar systems as nodes
   - Color-coded threat overlays based on recent kill events:
     RED = high danger (multiple kills in last hour)
     YELLOW = moderate activity
     GREEN = quiet/safe
   - Data source: KillmailCreatedEvent from the killmail module
   - Key fields: solar_system_id, killer_character_id, 
     victim_character_id, loss_type (SHIP or STRUCTURE), kill_timestamp

2. SMART ASSEMBLY MARKERS  
   - Icons on the map showing player-built structures in each system
   - Types: Gates, Turrets, Storage Units, Network Nodes
   - Shows owner, online/offline status
   - Data source: assembly objects from the blockchain gateway

3. PLAYER INTEL REPORTS (Sui wallet feature)
   - Players can pin a text warning to any solar system
   - e.g. "Fleet spotted here", "Ambush at this gate"
   - Reports are signed with the player's EVE Vault (Sui wallet)
   - Stored as lightweight Sui objects onchain — permanent and verifiable
   - Uses zkLogin for frictionless onboarding

4. GATE JUMP TRAFFIC
   - Shows which systems have high player movement
   - Helps identify busy trade routes vs. dead space
   - Data source: gate module jump events
